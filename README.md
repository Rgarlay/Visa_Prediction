# Visa Prediction Classification Using Machine Learning.

## Project Overview:

This project aims to predict the outcome of visa applications (Certified or Denied) using machine learning models. The dataset includes various features related to employment and applicant characteristics. The focus is on minimizing false negatives to ensure a high recall for the Certified class.

## Methods Used:

- Inferential Statistics
- Machine Learning
- Data Visualization
- Predictive Modeling


## Technologies Used: 
- Python, Sklearn, Pandas, Matplotlib
- HTML 
- Jupyter
